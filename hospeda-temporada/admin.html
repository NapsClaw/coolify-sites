Como editar o site

        

          📁 Estrutura dos arquivos:

          • index.html — página principal (lista de sítios)

          • sitio-iluminado.html — página do Sítio Iluminado

          • sitio-template.html — template usado pelos outros sítios

          • sitios-data.js — arquivo de dados (edite aqui para alterar nomes, fotos, preços)

          • admin.html — este painel


          🔧 Para trocar nomes/fotos dos sítios de exemplo:

          Abra sitios-data.js em qualquer editor de texto e edite os campos de cada sítio.


          ➕ Para adicionar novo sítio:

          Use a aba "Novo Sítio" acima — ela gera o código pronto para copiar no arquivo.
